# Linux-ის საბაზისო ბრძანებები

ყველა ბრძანება ამ თავში სრულდება Ubuntu/WSL ტერმინალში.

## მიმდინარე მდებარეობა

```bash
pwd
```

`pwd` აჩვენებს, რომელ ფოლდერში იმყოფებით.

## საქაღალდის შიგთავსის ნახვა

```bash
ls
ls -lh
ls -la
```

- `ls` — ფაილებისა და ფოლდერების სია;
- `-l` — დეტალური ინფორმაცია;
- `-h` — ფაილის ზომა ადვილად წასაკითხ ფორმატში;
- `-a` — დამალული ფაილების ჩვენება.

## ფოლდერში გადასვლა

```bash
cd folder_name
cd ..
cd ~
```

- `cd folder_name` — მითითებულ ფოლდერში გადასვლა;
- `cd ..` — ერთი დონით უკან;
- `cd ~` — home ფოლდერში დაბრუნება.

ფაილის ან საქაღალდის სახელში სივრცის არსებობისას გამოიყენეთ ბრჭყალები:

```bash
cd "My Folder"
```

ან სივრცე escape-ით:

```bash
cd My\ Folder
```

## საქაღალდისა და ფაილის შექმნა

```bash
mkdir results
touch notes.txt
```

რამდენიმე ჩადგმული საქაღალდის შესაქმნელად:

```bash
mkdir -p project/input project/output project/scripts
```

## კოპირება, გადატანა და სახელის შეცვლა

```bash
cp input.fasta backup.fasta
mv old_name.fasta new_name.fasta
mv file.fasta results/
```

საქაღალდის სრული კოპირება:

```bash
cp -r source_folder copied_folder
```

## ფაილის წაშლა

```bash
rm file.txt
rm -r folder_name
```

ფრთხილად: Linux-ში `rm`-ით წაშლილი ფაილი ჩვეულებრივ Recycle Bin-ში არ გადადის.

## ტექსტური ფაილის ნახვა

```bash
cat file.txt
head file.txt
tail file.txt
less file.txt
```

`less`-დან გამოსასვლელად დააჭირეთ `q`-ს.

## ტექსტური ფაილის რედაქტირება Nano-თი

```bash
nano file.txt
```

Nano-ში:

- შენახვა — `Ctrl+O`, შემდეგ `Enter`;
- გამოსვლა — `Ctrl+X`;
- ძიება — `Ctrl+W`.

## ბრძანების ისტორია

```bash
history
```

წინა ბრძანებების სანახავად ასევე გამოიყენეთ კლავიატურის ზედა და ქვედა ისრები.

## პროგრამის მდებარეობის შემოწმება

```bash
which mafft
command -v iqtree2
```

## მიმდინარე ბრძანების შეწყვეტა

```text
Ctrl+C
```

ეს ხშირად გამოიყენება ხანგრძლივი ან შეცდომით გაშვებული პროცესის შესაჩერებლად.

## Tab completion

ფაილის ან ბრძანების სახელის ნაწილის აკრეფის შემდეგ დააჭირეთ `Tab`-ს. Linux ავტომატურად შეავსებს სახელს ან შემოგთავაზებთ ვარიანტებს.
